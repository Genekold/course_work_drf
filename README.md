# Трекер привычек (Курсовая работа DRF/Docker)
***
## Описание проекта
***
### Проект представляет собой сервис для отслеживания полезных привычек. Пользователи могут создавать, отслеживать и делиться своими привычками, получать уведомления через Telegram.

## [Habit](https://51.250.44.120:80/swagger/) - это само приложение.

## Запуск проекта на любом сервере:

>1 Для начала необходимо настроить сервер для работы с Docker.
>>

>2 На сервере определить место для установки проекта 

>3 Клонировать репозиторий:
```bash
   git clone git@github.com:Genekold/course_work_drf.git
```
>4 На основе файла .env.example создать файл .env в корне проекта и заполнить своими данными

>5 Собрать и запустить контейнеры:
```bash
   docker-compose up -d --build
```
>6 Для остановки всех контейнеров и удаления контейнеров и томов выполните:
```bash
   docker-compose down -v
```

## Настройка CI/CD

### GitHub Actions

>1. Добавьте следующие секреты в настройки GitHub репозитория:

   - `SECRET_KEY` - секретный ключ
   - `DOCKER_HUB_ACCESS_TOKEN` - токен docker hub (write)
   - `DOCKER_HUB_USERNAME` - username в docker hub
   - `SERVER_IP` - IP вашего сервера
   - `SSH_PRIVATE_KEY` - приватный ssh ключ
   - `SSH_USER` - имя сервера

>2. При пуше и пул реквесте автоматически запускаются:
   - Тесты
   - Линтинг кода
   - Сборка Docker образов
   - Деплой на сервер
